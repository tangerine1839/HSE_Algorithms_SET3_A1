Программа в файле wide.cpp выводит площадь, если оценивать внутри большого прямоугольника, в narrow.cpp - внутри маленького прямоугольника.

Файл plot.py содержит код на python для построения графиков по данным из двух файлов (файлы так же приведены).

По первому графику видно, что чем больше точек используется для оценки площади, тем точнее она получается. Кроме того для маленького прямоугольника значения получились точнее - т.к. в нем плотность попадающих в пересечение точек выше.

По второму графику выводу такие же: отклонения становится меньше с увеличением количества точек, относительное отклонение для маленького прямоугольника снижается быстрее.
